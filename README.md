# 📊 Project Overview

This project presents an **interactive AI & Machine Learning Jobs Dashboard** built using **Power BI**.  
The dashboard analyzes global **AI job market trends**, including:

- 💰 Salary distribution
- 🌍 Location based hiring trends
- 🧠 Skills demand
- 🏢 Company hiring insights
- 👨‍💻 Job roles and experience levels

The goal of this project is to transform **raw job market data into meaningful insights** that help understand **AI industry hiring patterns and career opportunities**.

# 📂 Dataset Structure

The dataset consists of multiple relational tables:

### 1️⃣ Company Table
Contains information about companies hiring for AI jobs.

Columns include:

- Company Name
- Company Size
- Industry
- Company Rating

Example Companies:

- OpenAI  
- Google DeepMind  
- HuggingFace  
- NVIDIA  
- DataCamp  

---

### 2️⃣ Job Table

Contains job role information.

Columns include:

- Job Title
- Experience Level
- Salary
- Remote Ratio
- Employment Type
- Posted Date

Example Roles:

- Data Scientist  
- ML Engineer  
- NLP Engineer  
- Data Engineer  
- Computer Vision Engineer  
- AI Product Manager  

---

### 3️⃣ Skills Table

Contains technical skills required for each job.

Key Skills:

- Python  
- Machine Learning  
- Deep Learning  
- NLP  
- SQL  
- TensorFlow  
- PyTorch  
- Computer Vision  
- Cloud Computing  

---

### 4️⃣ Location Table

Contains job location information.

Cities included:

- San Francisco 🇺🇸  
- London 🇬🇧  
- Berlin 🇩🇪  
- Toronto 🇨🇦  
- Bengaluru 🇮🇳

# 📊 Dashboard Pages

The Power BI dashboard contains **4 main analytical pages**.

---

# 1️⃣ Overview Dashboard

This page provides **high-level job market insights**.

Key Metrics:

- Total Jobs
- Average Salary
- Median Salary
- Remote Job Percentage

Insights:

✔ Average AI salary ≈ **$93K**  
✔ Median salary ≈ **$102K**  
✔ Mid-level roles dominate the market  

---

# 2️⃣ Location Insights

This section analyzes **geographic distribution of AI jobs**.

Insights:

🌎 North America offers **highest salaries**  
🇪🇺 Europe has strong AI hiring demand  
🇮🇳 India is an **emerging AI hub**

Cities with highest job opportunities:

- San Francisco
- London
- Berlin
- Toronto
- Bengaluru

---

# 3️⃣ Company & Role Insights

This page analyzes **company hiring patterns and job roles**.

Insights:

🏢 Top companies hiring AI talent:

- OpenAI
- Google DeepMind
- NVIDIA
- HuggingFace

💼 Highest paying roles:

- Deep Learning Specialist
- ML Engineer
- Computer Vision Engineer

---

# 4️⃣ Skill & Hiring Insights

This page focuses on **skills required for AI jobs**.

Top skills demanded:

🥇 Python  
🥈 Machine Learning  
🥉 Deep Learning  
🏅 NLP  
🏅 Cloud Computing  

Insight:

Advanced AI roles require **specialized skills like Deep Learning and Computer Vision**.
